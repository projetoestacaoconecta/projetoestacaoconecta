Longa descrição do curso :
Descubra o mundo do desenvolvimento de websites com nosso curso introdutório, projetado especialmente para quem está começando. 
Neste curso, você aprenderá a estruturar páginas com HTML, estilizá-las com CSS e adicionar interatividade com JavaScript.

O curso conta com diversas experiencias como:
 - HTML: Você começará entendendo a estrutura básica de uma página web, utilizando HTML para criar elementos como cabeçalhos, parágrafos, links e imagens.
 - CSS: Em seguida, exploraremos como usar CSS para dar estilo ao seu site. Você aprenderá sobre cores, fontes, alinhamento e o layout de página, além de técnicas para fazer seu site se destacar visualmente.
 - JavaScript: Por fim, introduziremos conceitos básicos de JavaScript, a linguagem que traz interatividade ao seu site. Você aprenderá a manipular elementos HTML, responder a eventos do usuário e implementar lógica básica.
